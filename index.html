About

What is Archive Clanker?

Archive Clanker is a Discord bot that makes it easy to submit content to the Internet Archive without leaving Discord. It supports archiving websites, videos, git repositories, and entire wikis.


The bot can be added to any server or used directly from DMs and private channels — no server install required.

Under the Hood

  • tubeup + yt-dlp for video archiving
  • Save Page Now 2 API for web archiving
  • iagitbetter for git repositories
  • wikiteam3 for MediaWiki sites
  • dokuWikiDumper for DokuWiki
  • pukiWikiDumper for PukiWiki
