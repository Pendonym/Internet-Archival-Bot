Internet Archival is a Discord bot for archiving websites, videos, git repositories, and wikis to the Internet Archive.

Internet Archival uses tubeup & yt-dlp for video, the Save Page Now API for web archiving, iagitbetter for git repos, and wikiteam3 / dokuWikiDumper / pukiWikiDumper for wikis. Works in any server, DM, or private channel — no server installation required.

Bot Updates

  • /wiki-archive added — auto-detects MediaWiki, DokuWiki, PukiWiki
  • /web-archive now shows outlinks & embed counts
  • Playlist & channel support in /tubeup-archive
  • Per-item progress updates with DM fallback
  • HTTP 401/429 handling for Wayback Machine
